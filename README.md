# ATM Machine Simulation (Python)

##  Project Description

This is a simple ATM Machine Simulation project written in Python. The program allows users to:

* Insert an ATM card
* Verify the bank name
* Verify the card holder name
* Check account balance
* Deposit money
* Withdraw money
* Exit the system

This project was created to practice basic Python concepts such as:

* Variables
* Lists
* Conditional statements 
* User input 
* Match-case statements
* Basic menu-driven programs

## 🚀 Features

* Bank name verification
* Card holder authentication
* ATM menu options
* Balance checking
* Money deposit simulation
* Money withdrawal simulation
* Exit option

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Install Python 3 on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python atm.py
```
